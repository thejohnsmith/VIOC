Wireframe Notes

  1. ￼￼All details that pertain to an individual store will be editable from one long scroll page.
  2. Basic store details are pulled from the Data Mart and displayed to the user for reference. Helper text lets the user know where they need to go in order to edit it.
  3. Photo field, minimum resolution 350px x 120px. Store photo uses a default image when not set. When a new image is uploaded the user can zoom and adjust image to make sure it fits the frame well. (Recommend using a web based image editor like the Minimal Controls example on http://www.croppic.net)
